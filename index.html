SET LIST

We have an extensive set list and are well experienced in managing the set list either live or we can work with you to develop the best set arrangement, including specific songs for your event. We include background / iPad music during breaks as standard.

Example set times as standard cover 3 hours on site and we are flexible with no fuss in the arrangements. Some standard examples are 2x 60 minutes with a 20 min break or 3x 40 minutes with 15 min breaks. On average a set of 1 hour would cover 16-19 songs.
